# customer-churn-prediction
Machine Learning project to predict customer churn
## Project Overview: This Project predicts whether a customer will Churn (leave the company) or not using Machine Learning Models.
## Dataset
- Telecom customer Dataset
- Target Variable: Churn(Yes/No)
- -------
## Steps Performed
-Data Cleaning
-Exploratory Data Analysis(EDA)
-Encoding Categorical Variables
-Feature Scaling
-Model Building
-----------
## Models Used
-Logistic Regression
-Random Forest Classifier
-XGBoost Classifier
--------------
## Model Performance
-Logistic Regression Accuracy: ~76-77%
-Random Forest Accuracy:~80%
-XGBoost Accuracy:~81%
----------------
## Key Insights
-Tenure is the most important Feature
-Monthly Charges Affect Churn Significantly
-Contract type plays a major role
------------------
## Tools and Technologies
-Python
-Pandas,Numpy
-Scikit-learn
-Matplotlib,Seaborn
---------------------
## Conclusion
Machine Learning Models can effectively predict customer churn.
advanced models like Random Forest Classifier and XGBoost improved performance compared to Logistic Regression.
